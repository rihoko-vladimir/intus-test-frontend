# Intus-Frontend

Intus-Frontend is a part of the task for Intus Windows.

## Installation

Clone repository and use npm install.

```bash
npm i
```

## Running

Type ng serve. You need to have Intus-Backend running for application to work properly

```bash
ng serve
```
